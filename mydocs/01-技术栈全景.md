# 01 技术栈全景

## 1. 平台与语言
- **Python 3.11**：主开发语言。
- **Windows**：项目主要运行平台（窗口控制、输入模拟依赖 Win32 行为）。

## 2. GUI 与界面技术
### PySide6
- 库：`PySide6==6.8.0.2`
- 落点：`src/one_dragon_qt/**`、`src/zzz_od/gui/**`
- 用途：实现桌面窗口、页面、控件、信号槽事件。

### qfluentwidgets
- 库：`pyside6-fluent-widgets==1.7.0`
- 落点：`src/one_dragon_qt/widgets/**`
- 用途：提供 Fluent 风格导航、设置卡片、提示组件。

## 3. 输入控制与窗口交互
### pyautogui
- 库：`pyautogui==0.9.54`
- 落点：`src/one_dragon/base/controller/pc_controller_base.py`
- 用途：鼠标点击、拖拽、滚轮、键盘输入。

### pynput
- 库：`pynput==1.7.7`
- 落点：`keyboard_mouse_controller.py`、`pc_button_listener.py`
- 用途：
  - 模拟键盘/鼠标按键。
  - 监听全局按键触发开始/停止。

### ctypes（Win32 API）
- 库：Python 内置 `ctypes`
- 落点：`zzz_pc_controller.py`、`pc_game_window.py`、`backend_keyboard_mouse_contoller.py`
- 用途：窗口句柄操作、鼠标相对移动、消息发送等底层能力。

### vgamepad（可选）
- 库：`vgamepad==0.1.0`（`gamepad` 依赖组）
- 落点：`xbox_button_controller.py`、`ds4_button_controller.py`
- 用途：虚拟 Xbox / DS4 手柄输入。

## 4. 截图与图像处理
### mss
- 库：`mss==9.0.1`
- 落点：`pc_controller_base.py`
- 用途：高性能窗口截图。

### OpenCV
- 库：`opencv-python==4.10.0.84`
- 落点：`src/one_dragon/base/cv_process/**`、`src/zzz_od/**`
- 用途：阈值分割、轮廓分析、模板匹配、形态学处理等。

### numpy
- 落点：CV/OCR/YOLO 全链路
- 用途：图像矩阵计算、数据格式转换。

## 5. OCR 与 ONNX 推理
### onnxruntime-directml
- 库：`onnxruntime-directml==1.18.0`
- 落点：`onnx_model_loader.py`、`onnxocr/predict_base.py`
- 用途：运行 ONNX 模型（CPU / DirectML GPU）。

### onnxocr（仓库内 OCR 引擎）
- 落点：`src/onnxocr/**`
- 用途：文本检测（det）+ 方向分类（cls）+ 文本识别（rec）。

### OnnxOcrMatcher（项目封装）
- 落点：`src/one_dragon/base/matcher/ocr/onnx_ocr_matcher.py`
- 用途：OCR 模型下载、加载、缓存、识别接口统一。

## 6. YOLO 检测链
- 落点：`src/one_dragon/yolo/**`、`src/zzz_od/yolo/**`
- 用途：目标检测/分类及结果业务化封装。

## 7. 配置与数据驱动
### PyYAML
- 库：`pyyaml==6.0.1`
- 落点：`src/one_dragon/base/config/**`、`src/zzz_od/config/**`
- 用途：读取大量 YAML 参数与策略。

### 配置抽象
- 落点：`yaml_config.py`、`json_config.py`、`application_config.py`
- 用途：统一配置读写模式，减少业务层重复代码。

## 8. 任务编排与状态机
### Operation/Application 框架
- 落点：`src/one_dragon/base/operation/**`
- 用途：
  - `Operation`：步骤级自动化节点。
  - `Application`：功能级流程组织。
  - `ApplicationRunContext`：运行状态管理（运行/暂停/停止）。

### conditional_operation
- 落点：`src/one_dragon/base/conditional_operation/**`
- 用途：条件树 + 状态驱动执行，用于复杂战斗策略。

## 9. 通知与遥测
### 多渠道推送
- 落点：`src/one_dragon/base/push/channel/**`
- 用途：支持 Telegram、Discord、钉钉、企业微信、SMTP、Webhook 等。

### 遥测 Telemetry
- 落点：`src/zzz_od/telemetry/**`
- 用途：性能、错误、事件采集，附带隐私控制与脱敏。

## 10. 工程化工具链
### uv
- 文件：`pyproject.toml`、`uv.lock`
- 用途：依赖安装、环境创建、版本锁定。

### Ruff
- 文件：`pyproject.toml` 下 `tool.ruff`
- 用途：静态检查与格式规范。

### PyInstaller
- 文件：`deploy/*.spec`
- 用途：打包 Windows 可执行文件。

### GitHub Actions
- 文件：`.github/workflows/*.yml`
- 用途：自动测试、构建、签名、发布。

### pytest
- 文件：`tests/**`
- 用途：自动化回归测试。
